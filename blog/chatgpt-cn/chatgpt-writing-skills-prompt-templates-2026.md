## Prompt 写作的三个核心原则

在看具体模板前，先理解三个底层逻辑，否则再多模板也没用：

### 原则一：角色 + 任务 + 约束 = 好 Prompt

❌ 差的方式：`帮我写一封邮件`  
✅ 好的方式：`你是一位资深HR，帮我写一封拒绝候选人的邮件，语气专业但不失温度，控制在150字以内`

**三要素缺一不可：**
- **角色**：告诉 AI 扮演什么专家（影响文风和专业度）
- **任务**：具体说清楚要写什么（场景、对象、目的）
- **约束**：字数、格式、语气、禁止出现的内容

### 原则二：给示例比给描述强10倍

想让 AI 写出特定风格？直接粘一段你喜欢的原文，说"模仿这段的风格"，效果远超任何文字描述。

### 原则三：分步迭代，不要一次要求太多

不要让 AI 一步生成5000字的完整文章。先让它出大纲，确认后再逐段展开——质量会高出很多，你也更好掌控方向。

## 场景一：工作周报 / 日报

### 使用场景
每周必写，但写起来最让人头疼——要既不水、又不啰嗦，还得让老板看了舒服。

### Prompt 模板

```
你是一位职场写作专家。根据以下工作记录，帮我整理成一份专业的周报。

要求：
- 结构：本周完成事项 / 进行中的工作 / 下周计划 / 需协调事项
- 语气：简洁专业，用数字量化结果
- 字数：300字以内

工作记录：
[在这里粘贴你本周的工作流水账]
```

### 实际效果示例

**输入（流水账）：**
> 周一开了两个需求评审会，周三提测了登录模块，周四修了3个bug，周五做了竞品分析报告，下周要做用户调研

**ChatGPT 输出（周报）：**
> **本周完成：**
> - 完成登录模块开发并提测，修复关联Bug 3个，测试通过率100%
> - 完成竞品分析报告，涵盖X家主要竞品，已共享至项目文档
> - 参与需求评审会议2次，对齐Q2核心功能规划
>
> **下周计划：**
> - 开展用户调研（目标样本：N人），输出调研报告
>
> **需协调：** 用户调研问卷设计需产品侧确认后启动

## 场景二：商务邮件

### 使用场景
催款、拒绝需求、向大客户汇报、内部协调——每封邮件都要拿捏分寸，写错了影响职业关系。

### Prompt 模板

```
你是一位有10年经验的商务写作专家。帮我写一封邮件：

发件人：[你的身份/部门]
收件人：[对方身份/关系]
目的：[这封邮件要达成什么目标]
关键信息：[必须包含的内容要点]
语气：[正式/友好/强硬/委婉]
字数限制：[XXX字以内]

特别注意：[有什么禁忌或特殊要求，没有可删掉这行]
```

### 技巧
催款邮件很难写？把"目的"改为"委婉催促对方尽快付款，同时保持良好合作关系"，AI 会自动帮你找到那个微妙的平衡点。

## 场景三：会议纪要

### 使用场景
开完会录音一堆，整理成纪要又烦又费时。

### Prompt 模板

```
帮我将以下会议记录整理成标准会议纪要。

要求格式：
- 会议基本信息（时间/参会人/主题）
- 讨论要点（分条列出，保留关键数据）
- 决策结论（明确写出达成的共识）
- 行动项（人名 + 事项 + 截止时间格式）

会议记录原文：
[粘贴录音转文字或手记流水账]
```

## 场景四：方案 / 提案撰写

### 使用场景
给领导写汇报方案、写项目提案、写营销计划——最耗时、最难起笔的场景。

### Prompt 模板（分两步走）

**第一步：出大纲**
```
你是一位资深咨询顾问。帮我制定一份[主题]的方案大纲。

背景：[项目/需求的背景情况]
目标受众：[谁来看这份方案]
核心诉求：[方案最需要解决什么问题]
特殊限制：[时间/预算/资源约束]

请先给我一个包含5-7个章节的大纲，每章用2-3行说明要写什么。
```

**第二步：逐章展开**
```
大纲确认，现在帮我展开第[X]章：[章节标题]

要求：
- 字数：[XXX]字左右
- 包含：[数据支撑/案例/图表建议]
- 风格：[说服性/分析性/汇报型]
```

## 场景五：社交媒体文案

### 使用场景
微信公众号、小红书、朋友圈、微博——每个平台风格完全不同，同一个内容要改四遍。

### Prompt 模板

```
将以下内容改写为[平台名]风格的文案：

原始内容：[产品介绍/活动信息/观点]

[平台名]风格要求：
- 小红书：口语化、第一人称、种草感、多用emoji、结尾带话题标签
- 公众号：有深度、有故事性、适合碎片化阅读、段落短
- 朋友圈：真实感、生活化、控制在100字以内、不能太广告
- 微博：简洁有力、带话题标签、适合转发

请输出[X]个不同版本供选择。
```

## 场景六：论文 / 报告润色

### 使用场景
写完的论文语言生硬？学术报告读起来像流水账？

### Prompt 模板

```
你是一位学术写作专家，擅长中文学术论文润色。请对以下段落进行改写：

改写要求：
1. 保持原意不变，不要改动数据和论点
2. 提升语言的学术性和流畅度
3. 避免口语化表达
4. 确保逻辑连接词使用规范

原文：
[粘贴你的段落]
```

### 进阶用法：降重

```
在不改变核心意思的前提下，对以下内容进行同义替换和句式重构，使其与原文相似度降低，同时保持学术语言规范：

[粘贴内容]
```

## 场景七：产品文案 / 电商描述

### 使用场景
写卖点、写详情页、写标题——需要既抓眼球又有转化力。

### Prompt 模板

```
你是一位电商运营文案专家。帮我为以下产品写卖点文案：

产品名称：[XXX]
目标用户：[年龄/性别/使用场景]
核心卖点：[列出3-5个实际功能/优势]
竞品对比：[我们比竞品强在哪里]

需要输出：
1. 主标题（15字以内，突出核心卖点）
2. 副标题（30字以内，补充说明）
3. 3个卖点短句（每条20字以内）
4. 100字详情页开头段
```

## 场景八：演讲稿 / 发言稿

### 使用场景
年会发言、团队分享、毕业典礼——需要有感情、有金句、有节奏。

### Prompt 模板

```
你是一位擅长演讲稿写作的专家。帮我写一篇演讲稿：

场合：[年会/分享会/颁奖/毕业典礼]
发言者身份：[你是谁，什么角色]
时长：[X分钟，换算约XXX字]
核心主题：[这次发言最想传达什么]
情感基调：[激励/感谢/总结/展望]
必须提到的内容：[具体人名/事件/数据]

结构要求：
- 开场30秒要抓住注意力
- 中间部分有1-2个真实故事或细节
- 结尾有让人记住的金句
```

## 场景九：简历 / 自我介绍

### 使用场景
工作经历写得干巴巴，看不出价值；自我介绍总是说不到重点。

### Prompt 模板

```
你是一位资深HR顾问。帮我优化以下工作经历描述，使其更有说服力：

目标职位：[你要应聘的岗位]
原始描述：[你现在简历上写的内容]

优化要求：
1. 用"动词+数字+结果"结构改写
2. 突出与目标职位相关的技能
3. 每条不超过30字
4. 输出3个版本供选择
```

## 场景十：读书笔记 / 内容总结

### 使用场景
看完一篇长文、一本书，需要快速提炼精华。

### Prompt 模板

```
请对以下内容进行结构化总结：

总结框架：
1. 核心论点（3条以内）
2. 关键论据或数据
3. 实操建议（如有）
4. 一句话精华（用于收藏/分享）

原文：
[粘贴文章/书摘]
```

## 进阶技巧：让输出质量再上一个台阶

### 技巧1：要求 AI 自我检查

在 Prompt 结尾加一句：
```
完成后，请检查：1）是否有冗余内容 2）逻辑是否流畅 3）是否符合要求，并主动修改。
```

### 技巧2：让 AI 扮演审稿人

写完后再发一条：
```
现在以严格的编辑身份审阅上面这段文字，找出3个可以改进的地方，并直接修改。
```

### 技巧3：用对比法选最优版本

```
给我写3个不同风格的版本：A版保守稳重，B版活泼有趣，C版数据驱动，我来选择最适合的。
```

## 常见问题

### Q：Prompt 越长越好吗？
不是。关键是**信息密度**，不是字数。一个包含角色、任务、约束、示例的100字 Prompt 比500字的废话堆砌效果好得多。

### Q：AI写出来的能直接用吗？
**70%左右可以直接用**，剩下30%需要人工调整——主要是加入只有你知道的具体细节。把 AI 当"高速草稿机"，不是"全自动写手"。

### Q：用哪个模型写作效果最好？
日常写作：**GPT-5.4** 和 **Claude 4.6** 不相上下。长文章和需要大量细节推理的内容，Claude 4.6 稍好；创意文案和营销内容，GPT-5.4 更有爆发力。[GPTCat](https://gptcat.cc) 两个模型都可以切换，可以各试一下。
