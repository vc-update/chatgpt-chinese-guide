# fill.py 已被禁用 / DISABLED

**禁用时间**：2026-05-11
**原因**：该脚本会遍历 `.vitepress/config.mts` 中所有 `text/link` 配置，并**强制覆盖**对应的 `.md` 文件为"🚧 内容正在复刻中..."占位模板。

## 危险性

如果在自动化流程或误操作中执行：
- 全站所有页面（>100 篇）会**瞬间变成空内容**
- Bing/Google 抓到大量"空壳页"后，整站会被判定为低质量，**触发站点级别降权**
- 已有的搜索排名将在 1-2 周内全部消失，可能数月无法恢复

## 如需重新启用

确认是否真的需要"批量重置内容"（绝大多数情况下答案是"否"）：

```bash
# 重命名回来
mv fill.py.DISABLED fill.py
# 慎用，最好先备份整个 chatgpt/ 目录
```

如果只是想给某个新页面填占位内容，**手动新建一个 .md 文件即可**，不要跑这个脚本。
