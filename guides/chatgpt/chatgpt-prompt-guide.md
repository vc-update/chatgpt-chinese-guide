# ChatGPT Prompt 教程（2026最新）

🕒 最后更新时间：2026-03

很多人在使用 ChatGPT 时会发现：

**同样的问题，不同的人得到的回答质量差别很大。**

原因通常不是 AI，而是 **Prompt（提示词）写法不同**。

Prompt 指的是用户输入给 AI 的指令。  
一个好的 Prompt 可以让 ChatGPT 给出更加精准、结构清晰、逻辑完整的回答。

本指南将为你详细介绍：

- 什么是 Prompt

# 一、什么是 ChatGPT Prompt

Prompt（提示词）是用户向 AI 输入的指令。

例如：

```
写一篇关于人工智能发展的文章
```

这就是一个 Prompt。

但如果你这样写：

```
你是一名科技作者，请写一篇1200字文章，主题是人工智能的发展趋势，并包含三个小标题。
```

AI 的回答质量通常会 **明显提高**。

因此：

**Prompt 的质量直接决定 AI 输出质量。**

---

# 二、Prompt 的核心公式

很多 AI 使用者会使用一个经典 Prompt 公式：

```
角色 + 任务 + 要求 + 背景
```

例如：

```
你是一名专业科技作者，请写一篇关于人工智能未来趋势的文章，字数约1200字，语言通俗易懂，并包含三个小标题。
```

解释：

**角色**

让 AI 扮演某个角色，例如：

- 科技作者
- 软件工程师
- 学术研究员
- 营销专家

**任务**

明确需要完成的事情。

**要求**

限制输出形式，例如：

- 字数
- 风格
- 格式

**背景**

提供更多信息，让 AI 理解上下文。

# 三、写文章 Prompt 模板

ChatGPT 最常见的用途之一就是 **写文章**。

## 模板1：博客文章

```
你是一名专业博客作者，请写一篇关于【主题】的文章，字数约1000字，语言通俗易懂，并包含三个小标题。
```

示例：

```
你是一名科技博客作者，请写一篇关于人工智能未来趋势的文章，字数1200字，并包含三个小标题。
```

适用场景：

- SEO文章
- 自媒体内容
- 博客文章

---

## 模板2：社交媒体文案

```
请为【产品】写一篇社交媒体文案，语气轻松活泼，并包含3个emoji。
```

示例：

```
请为一款无线耳机写一篇小红书文案，语气轻松，并包含emoji。
```

---

## 模板3：新闻稿

```
你是一名新闻编辑，请为【事件】撰写一篇新闻稿，语言客观正式，约800字。
```

# 四、编程 Prompt 模板

ChatGPT 在编程领域非常强大。

## 模板1：写代码

```
你是一名资深软件工程师，请使用【编程语言】编写一个【功能】程序，并附带代码解释。
```

示例：

```
你是一名资深软件工程师，请使用Python写一个简单的网页爬虫程序，并解释代码。
```

---

## 模板2：代码调试

```
请帮我分析以下代码，并找出可能存在的错误，同时给出优化建议。
```

---

## 模板3：学习编程

```
请用简单易懂的方式解释Python中的装饰器，并给出三个示例。
```

---

# 五、论文写作 Prompt 模板（重点）

很多学生和研究人员会使用 ChatGPT 辅助论文写作。

## 模板1：论文大纲

```
你是一名学术研究员，请为题为【论文主题】的论文生成一个详细大纲，包括研究背景、研究方法和研究结论。
```

示例：

```
你是一名学术研究员，请为题为“人工智能在医疗领域的应用”的论文生成一个详细大纲。
```

---

## 模板2：文献综述

```
请总结【研究领域】的主要研究进展，并列出3个关键研究方向。
```

示例：

```
请总结人工智能在医学影像分析领域的主要研究进展。
```

---

## 模板3：论文润色

```
请帮我润色以下论文段落，使其语言更加学术化，并保持原意。
```

学生也可以利用 ChatGPT 学习。

## 概念解释

```

```

# 八、提高 Prompt 质量的技巧

如果你想让 ChatGPT 给出更高质量的回答，可以尝试以下方法。

### 提供更多细节

不要只写：

```
写一篇文章
```

而是写：

```
写一篇1200字文章，主题是人工智能的发展趋势。
```

---

### 分步骤提问

例如：

第一步：

```
帮我生成文章大纲
```

第二步：

```
根据大纲写完整文章
```

---

### 要求改写

例如：

```
请把这段内容改得更简洁
```

或者：

```
请用更专业的语言重新表达
```
