## 四、Claude 4.6 使用技巧：发挥真实水平的 3 个 Prompt 技巧

很多人拿到 Claude 之后，只会输入简单的问题，结果觉得"也不过如此"。其实 Claude 对指令的响应非常细腻，换个方式问，效果可以差很多。

### 技巧 1：告诉它你"不想要什么"

大多数人只告诉 AI 要写什么，却忘了说"不要写什么"。Claude 对负向指令的理解极好：

```
帮我写一段产品介绍，不要用"在当今时代"开头，不要出现"赋能"、"生态"这类词，语气要像朋友推荐，不要像广告文案。
```

### 技巧 2：给它一个"身份"和"读者"

```
你是一位有十年经验的财经记者。现在你需要为一批从不看财经新闻的普通读者，解释"美联储降息"对他们日常生活的影响。用3个具体的生活场景来说明。
```

### 技巧 3：让它"先思考再输出"

对于复杂问题，可以让 Claude 在正式回答前先梳理思路：

```
在写正文之前，先列出你打算回答这个问题的逻辑框架（用3-5个要点），确认后我再让你展开写。
```

这个技巧在处理长文档、复杂分析时尤其有效，能大幅降低"跑题"的概率。
