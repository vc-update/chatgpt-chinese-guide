# 一、ChatGPT 是什么？

ChatGPT 是由 **OpenAI** 开发的一款人工智能聊天机器人，它基于大型语言模型（LLM）技术，可以理解并生成自然语言。

简单来说，你可以把 ChatGPT 看作一个 **智能助手**。

它可以帮助你完成很多任务，例如：

- 写文章
- 编程
- 翻译
- 学习新知识
- 制定计划

目前 ChatGPT 支持多个模型，例如：

- GPT-4
- GPT-4o
- GPT-5

这些模型在语言理解和内容生成方面都非常强大。

# 二、如何开始使用 ChatGPT

使用 ChatGPT 非常简单，只需要三步。

## 第一步：打开 ChatGPT

官方入口：

https://chatgpt.com

如果官网访问困难，可以使用国内入口：

https://snakegpt.vip

---

## 第二步：输入问题

在聊天框中输入你的问题，例如：

```
帮我写一篇关于人工智能发展的文章
```

ChatGPT 会在几秒钟内生成回答。

---

## 第三步：继续对话

ChatGPT 支持连续对话。

例如：

```
帮我把文章改成更简短的版本
```

AI 会根据上下文继续优化回答。

# 四、ChatGPT 提问技巧（Prompt）

很多新手在使用 ChatGPT 时会发现一个问题：

**为什么 AI 回答质量不稳定？**

原因通常是 **提问方式不够清晰**。

一个好的 Prompt 通常包含：

```
角色 + 任务 + 要求 + 背景
```

示例：

你是一名旅行规划师，请为我设计一个5天东京旅行计划，预算经济，并包含动漫景点。

这样 AI 的回答会更准确。
