## 第四步：GPT-5.4 基础用法——日常对话

### 4.1 问答型对话

最基础的用法就是**有问有答**：

```
你：中国有多少个省级行政区？分别是什么？
AI：中国共有34个省级行政区，包括23个省、5个自治区、4个直辖市和2个特别行政区...
```

### 4.2 任务型对话

ChatGPT真正的价值在于**帮你干活**。给它明确的任务和要求：

**写邮件：**
```
请帮我写一封商务邮件：
- 收件人：客户张经理
- 目的：邀请参加下周三的产品发布会
- 语气：正式但友好
- 包含：时间地点、议程概要、回复确认
```

**翻译：**
```
请将以下内容翻译为英文，保持商务语气：
"我们很高兴地通知您，贵公司已通过本轮供应商资质审核。"
```

**整理数据：**
```
以下是本月销售数据，请按地区汇总并计算环比增长率：
[粘贴你的数据]
```

### 4.3 上下文对话

ChatGPT会记住**当前对话的上下文**。你可以在一个对话中层层深入：

```
你：什么是机器学习？
AI：[解释机器学习的概念]

你：它和深度学习有什么区别？
AI：[基于上文对比两者区别]

你：用一个生活中的例子帮我理解深度学习
AI：[用日常案例解释]
```

> 📌 **技巧：** 一个话题开一个新对话。不要在同一个对话里混着聊完全不同的主题，不然AI的上下文会混乱。

## 第五步：解锁GPT-5.4三大高级功能

基础用法学会了，下面是真正让GPT-5.4区别于其他AI的**杀手级功能**。

### 5.1 Thinking深度推理模式

**什么是Thinking？** 开启后，GPT-5.4会在回答前先进行一轮内部推理，你能看到它"思考"的过程（一个展开的思维链）。

**什么时候用？**
- 数学题/逻辑题
- 复杂编程Bug调试
- 需要多步推理的分析任务
- 策略规划/决策分析

**实操示例：**

```
[开启Thinking模式]

你：一个水池有两个进水管和一个出水管。A管单独注满需要6小时，B管单独注满需要4小时，
出水管单独放空需要12小时。三管同时打开，多久能注满水池？

AI：
[思考过程]
A管每小时注入1/6，B管每小时注入1/4，出水管每小时流出1/12...
合计每小时净注入 = 1/6 + 1/4 - 1/12 = 2/12 + 3/12 - 1/12 = 4/12 = 1/3
所以需要3小时注满。

[最终答案]
三管同时打开，3小时注满水池。
```

> 在[GPTCat](https://gptcat.cc)上选择GPT-5.4模型后，可以在输入框旁切换到Thinking模式。

### 5.2 Deep Research深度调研

**什么是Deep Research？** 它不是简单搜索——GPT-5.4会**自动执行多轮网络搜索**，阅读数十个网页，交叉验证信息，最后生成一份带引用的深度报告。

**什么时候用？**
- 行业分析/市场调研
- 竞品对比研究
- 学术文献综述
- 投资标的分析

**实操示例：**

```
请帮我做一个深度调研：
2026年中国AI大模型市场格局分析，包括：
1. 主要玩家及其最新产品
2. 各家技术路线对比
3. 商业化进展
4. 未来6个月趋势预判
```

GPT-5.4会花3-5分钟自动搜索、阅读、分析，最后输出一份3000-5000字的结构化报告。这种质量的报告，手动做至少需要一整天。

### 5.3 多模态交互

GPT-5.4支持**图片、语音、视频**输入：

**看图分析：**
- 拍一张菜谱照片 → 自动识别食材和做法
- 截图一段代码报错 → 自动分析错误原因
- 上传一张图表 → 自动提取数据和趋势

**语音对话：**
- 像打电话一样和AI聊天
- 支持中英文混合对话
- 适合练英语口语、听写训练

> [GPTCat](https://gptcat.cc)完整支持语音对话和视频理解，在对话界面点击麦克风或上传按钮即可使用。

## 第六步：20个即学即用的Prompt模板

好的Prompt（提示词）是用好ChatGPT的关键。以下模板覆盖最常用的场景，**直接复制、填上你的内容就能用**。

### 📝 写作类

**1. 公众号文章**
```
主题：[你的主题]。请写一篇2500字的公众号文章，要求：
开头用故事或数据引入，正文分3-4个核心观点，每个观点配一个案例，
结尾有行动建议。语气专业但易懂，适合[目标读者]阅读。
```

**2. 小红书笔记**
```
写一篇小红书笔记，主题：[主题]。
要求：5个标题备选（含emoji），正文800字，语气口语化亲和，
段落用emoji分隔，结尾引导互动，附10个标签。
```

**3. 论文润色**
```
请对以下学术段落进行润色，要求：
保持原意，使用正式学术语气，优化句式结构降低重复率，
确保逻辑连贯。如有表述不当之处请标注说明。
[粘贴段落]
```

### 💻 编程类

**4. 代码生成**
```
用[语言]实现以下功能：[详细描述]
要求：代码结构清晰，包含注释，有完善的错误处理，
附使用示例和运行说明。
```

**5. Bug调试（建议开Thinking模式）**
```
我的代码遇到以下错误：
代码：[代码]
报错：[报错信息]
预期行为：[应该发生什么]
实际行为：[实际发生了什么]
请分析原因并给出修复方案。
```

**6. 代码审查**
```
请审查以下代码，从性能、安全性、可读性、最佳实践四个角度给出具体改进建议：
[代码]
```

### 📊 数据分析类

**7. 数据解读**
```
以下是[数据描述]的数据：
[粘贴数据/表格]
请分析：1)整体趋势 2)异常值及原因猜测 3)关键指标变化 4)3-5条建议
```

**8. SQL生成**
```
数据库有以下表：[描述表结构]
请写SQL实现：[你想查询的内容]
要求：附注释说明每段查询逻辑。
```

### 🎓 学习类

**9. 概念解释（费曼学习法）**
```
请用费曼学习法解释"[概念]"：
1)先用小学生能懂的话解释 2)逐步深入到专业层面
3)用一个生活类比 4)给3个思考题检验我的理解
```

**10. 学习路线**
```
我想从零学[技能]，每天[X小时]，目标[X个月]达到[水平]。
请制定详细学习路线：分阶段目标+推荐资源+实践项目+里程碑。
```

### 🏢 办公类

**11. 商务邮件**
```
写一封商务邮件，收件人：[身份]，目的：[目的]，
语气：[正式/友好]，包含要点：[列举]。
```

**12. 会议纪要**
```
以下是会议文字稿，请提炼为会议纪要：
格式：1)会议信息 2)讨论要点 3)决议事项 4)待办（含责任人+截止日）
[文字稿]
```

**13. PPT大纲**
```
主题：[主题]，受众：[谁看]，时长：[X分钟]。
生成PPT大纲：包含封面、目录、每页标题+3-4个要点+演讲备注。
```

### 🌐 翻译类

**14. 精准翻译**
```
将以下内容翻译为[目标语言]：
语气：[正式/口语/商务]，保留专业术语原文标注，
标注文化差异需调整之处。
[原文]
```

**15. AI绘图提示词**

## 第七步：避坑指南——新手常犯的5个错误

### ❌ 错误1：问题太模糊

```
差：帮我写一篇文章
好：帮我写一篇2000字的科普文章，主题是"量子计算对普通人意味着什么"，
   目标读者是非技术背景的白领，语气轻松有趣，包含3个日常生活的类比
```

### ❌ 错误2：一次塞太多任务

把大任务拆成小步骤，一步步来。先让AI生成大纲，确认后再逐章展开。

### ❌ 错误3：不核实关键信息

AI有时会"一本正经地胡说八道"。涉及数据、法律、医疗等关键信息，**一定要交叉验证**。

### ❌ 错误4：不利用上下文

同一个对话里，你可以要求"改短一点""语气更正式""加一个表格"——AI会基于上下文修改，不用从头再来。

### ❌ 错误5：只用一个模型

不同模型擅长不同任务。推理用GPT-5.4 Thinking，长文用Claude 4.6，中文创作用DeepSeek-V3。在[GPTCat](https://gptcat.cc)或[SnakeGPT](https://snakegpt.vip)上一键切换。
