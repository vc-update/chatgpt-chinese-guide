## 三、ChatGPT使用教程：镜像站 5 分钟快速上手（推荐）

以 GPTCat 为例，演示从注册到发出第一条对话的完整流程。

### 步骤 1：访问平台

打开浏览器（推荐 Chrome / Edge），直接输入 `https://gptcat.cc`，国内网络可直接打开，无需任何代理。

### 步骤 2：注册账号

1. 点击右上角 **"注册"** 按钮
2. 输入国内邮箱（QQ邮箱 / 163邮箱 / Gmail 均可）
3. 设置密码（建议 8 位以上，含字母 + 数字 + 符号）
4. 点击 **"发送验证码"**，查收邮件
5. 输入 6 位验证码，点击 **"完成注册"**

> 💡 小贴士：若未收到邮件，请检查"垃圾邮件"文件夹，或换用 Gmail 重新注册。

### 步骤 3：登录并选择模型

登录后进入聊天界面，点击对话框上方的模型选择器：

| 模型 | 适用场景 |
|------|---------|
| **GPT-5.4** | 复杂推理、专业写作、代码 |
| **Claude 4.6** | 长文本分析、细腻创作 |
| **Gemini 3.1 Pro** | 数据分析、多模态任务 |
| **Grok 4.2** | 实时资讯、社交热点 |
| **DeepSeek-V3** | 中文场景、国产优化 |
| **MidJourney** | AI 绘图 |

### 步骤 4：发送第一条对话

在底部输入框输入你的问题并按 Enter 发送。新手测试 Prompt 推荐：

```
你好，请用简洁的语言介绍你自己，并列出你最擅长帮我完成的 5 件事。
```

### 步骤 5：解锁进阶功能

- **📎 上传文件**：点击输入框左侧回形针图标，支持 PDF / Word / Excel / 图片
- **🎤 语音输入**：点击麦克风图标，支持连续语音转文字
- **🎨 AI 绘图**：切换 MidJourney 模型，输入画面描述即可
- **🗂️ 新建会话**：点击左侧 "新建对话"，开启独立上下文
- **📌 对话收藏**：长按对话可收藏或导出 Markdown

## 四、ChatGPT 官网注册教程（进阶用户）

若你坚持使用 ChatGPT 官网原生服务，2026 年 05 月最新完整流程如下。

### 前置准备清单

- [ ] 稳定代理工具（节点选美国 / 欧洲 / 日本）
- [ ] Gmail 或 Outlook 邮箱
- [ ] 海外手机号（可在 sms-activate.org 等接码平台获取）
- [ ] 浏览器开启无痕模式并清除 Cookie

### 完整步骤

1. **开启代理**，确认当前 IP 为境外
2. **访问** `https://chat.openai.com`，点击右上角 **Sign up**
3. **选择注册方式**：邮箱 / Google / Microsoft / Apple（推荐邮箱）
4. **填写邮箱 + 密码**，提交后前往邮箱点击 `Verify email address`
5. **填写姓名** + 生日（年龄需 ≥ 18）
6. **验证手机号**：选择国家（避免选中国 +86），填写海外号，接收验证码
7. **完善资料**，同意服务条款，进入主界面
8. **升级 Plus**（可选）：解锁 GPT-5.4、o3、DALL-E 3、语音等完整功能

> ⚠️ 注意：若提示 "Access Denied" 或 "Country not supported"，请更换代理节点并清除浏览器缓存后重试。

## 五、ChatGPT 实战 Prompt 技巧（进阶必学）

掌握 Prompt 的写法是决定 AI 输出质量的核心。以下是 2026 年最实用的 5 大 Prompt 技巧：

### 技巧 1：角色扮演法（Role Prompting）

```
你是一位有10年经验的资深 Python 工程师，请帮我 Review 以下代码并指出性能瓶颈：
[贴上代码]
```

### 技巧 2：分步骤指令（Chain of Thought）

```
请分三步解答：
1. 先分析问题背景
2. 列出可能的解决方案
3. 推荐最佳方案并说明理由
```

### 技巧 3：格式化输出

```
请用 Markdown 表格输出，包含以下列：功能名称 / 使用场景 / 注意事项
```

### 技巧 4：提供示例（Few-shot）

```
按以下风格改写我的文案：
示例1：原文→改写
示例2：原文→改写
现在请改写：[你的文案]
```

### 技巧 5：深度思考指令

```
请启用深度思考模式，先列出思考过程，再给出最终答案。
```
