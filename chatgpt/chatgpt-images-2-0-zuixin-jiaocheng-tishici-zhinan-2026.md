## 三、完整使用教程（5步上手）

### 步骤 1：选择访问方式

**方式一：ChatGPT 官网**
- 打开 [chatgpt.com](https://chatgpt.com) 或 [chatgpt.com/images](https://chatgpt.com/images)
- 需要海外网络环境 + 账号

**方式二：国内平台（推荐）**
- 打开 [GPTCat](https://gptcat.cc) 或 [SnakeGPT](https://snakegpt.vip)
- 国内邮箱注册即可，无需处理网络问题
- 完整支持 GPT Image 2 所有功能

### 步骤 2：进入图像生成

在对话框中直接输入图像相关指令即可触发生图。也可以点击输入框左侧「**+**」→ 选择「**创建图片**」。

### 步骤 3：编写提示词

提示词是决定出图质量的关键。基本结构：

```
[主题] + [风格] + [构图/布局] + [色调/氛围] + [文字内容（如需要）] + [尺寸/质量要求]
```

### 步骤 4：选择生成模式

- 简单图片 → 直接发送（Instant 模式）
- 复杂海报/中文内容 → 在 Prompt 中加"请使用思考模式生成"

### 步骤 5：编辑与优化

生成后点击图片进入编辑面板：
- 调整宽高比（适配不同平台）
- 局部修改（上传图片 + 修改指令）
- 批量生成（Prompt 加"生成 4 个不同版本"）

## 五、提示词进阶技巧

掌握以下技巧，出图质量再上一个台阶：

### 技巧 1：结构化提示词公式

```
[主体描述] + [风格关键词] + [构图指令] + [光影/色调] + [文字要求] + [技术参数]
```

**示例：**
```
一位穿着汉服的少女站在樱花树下（主体），水彩插画风格（风格），半身特写构图（构图），逆光柔和，粉白色调（光影），无文字（文字），高清细腻（参数）
```

### 技巧 2：中文文字渲染要点

1. 用引号明确标注要出现的文字：`标题文字为"2026 AI指南"`
2. 指定字体风格：`清晰黑体、无衬线、加粗`
3. 指定文字位置：`标题居中放在画面上方 1/3 处`
4. 开启 Thinking 模式（中文渲染质量显著更高）

### 技巧 3：批量生成系列图

在 Prompt 末尾加上：
- `生成 4 个不同版本`（同主题不同构图）
- `生成一组风格统一的系列图（4张）`（保持一致性）
- Thinking 模式下最多可生成 8 张高度一致的图像

### 技巧 4：图片编辑与迭代

生成后可以继续对话优化：
- `把背景换成深蓝色`
- `文字再大一些，放到画面中央`
- `保持整体风格不变，换一个角度`
- `上传一张参考图` + `按这个风格重新生成`

### 技巧 5：指定宽高比

不同平台需要不同尺寸：
| 平台 | 推荐比例 | Prompt 写法 |
|------|---------|------------|
| 公众号封面 | 2.35:1 | `横版宽幅，比例 2.35:1` |
| 小红书 | 3:4 | `竖版，比例 3:4` |
| 抖音封面 | 9:16 | `竖版全屏，比例 9:16` |
| PPT配图 | 16:9 | `横版，比例 16:9` |
| 头像/Logo | 1:1 | `正方形构图` |

## 七、常见问题 FAQ

**Q1：免费用户能用 ChatGPT Images 2.0 吗？**

可以。所有用户都能使用 Instant 模式，每日 5-10 张额度。体验够用，高频使用建议升级 Plus 或用国内平台。

**Q2：怎么开启 Thinking 思考模式？**

Plus / Pro / Business 用户在模型选择中切换到 Thinking 模式，或在 Prompt 中说"使用思考模式生成"。Thinking 模式下中文文字和复杂排版质量显著更高。

**Q3：生成的中文还是模糊怎么办？**

四个解决方法：
1. 用引号明确写出要出现的文字
2. 指定字体风格（如"清晰黑体、无衬线"）
3. 开启 Thinking 模式
4. 生成后追加指令："优化所有文字清晰度和排版"

**Q4：国内用户怎么用？**

推荐 [GPTCat](https://gptcat.cc) 或 [SnakeGPT](https://snakegpt.vip)，国内邮箱注册，完整支持 GPT Image 2，无需处理网络问题。

**Q5：生成的图片能商用吗？**

根据 OpenAI 使用条款，付费用户生成的图片可用于商业用途。正式商用前建议核对官方最新政策。

**Q6：一次能生成多少张？**

Instant 模式单次 1 张；Thinking 模式下 Prompt 加"生成 N 个版本"，最多 8 张风格一致的系列图。

**Q7：可以上传参考图吗？**

可以。上传图片后输入修改指令，如"按这个风格重新生成""把人物换成卡通风格""保留构图，换成水彩风格"等。

**Q8：和 Midjourney 比怎么样？**

GPT Image 2 优势在于：中文支持好、对话式交互（可迭代修改）、指令遵循精准。Midjourney 优势在于：纯艺术风格化更强、社区生态成熟。日常实用场景推荐 GPT Image 2。
