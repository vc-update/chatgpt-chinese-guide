## 四、ChatGPT基础操作：新手必学的5个技能

### 技能1：写出好提示词（Prompt）

ChatGPT的输出质量很大程度取决于你怎么"问"。好的提示词应包含：

| 要素 | 差的示例 | 好的示例 |
|------|---------|---------|
| **角色设定** | （无） | "你是一名10年经验的产品经理" |
| **具体任务** | "帮我写文章" | "帮我写一篇500字的产品发布公告" |
| **背景信息** | （无） | "产品是一款面向中小企业的SaaS工具" |
| **输出格式** | （无） | "用标题+正文+行动号召三段式结构" |
| **约束条件** | （无） | "语气专业但不生硬，避免使用专业术语" |

**实战示例：**

```
你是一名资深HR，请帮我写一封拒绝候选人的邮件。
候选人参加了我们的产品经理面试，综合能力不错但经验不足。
要求：语气委婉、专业，保留未来合作可能，200字以内。
```

### 技能2：多轮对话与追问

ChatGPT支持上下文记忆，同一对话窗口内你可以不断追问：

```
用户：给我讲一下机器学习的基本概念
ChatGPT：[回答基本概念]

用户：能举个生活中的具体例子吗？
ChatGPT：[结合上下文举例]

用户：那深度学习和机器学习有什么区别？
ChatGPT：[基于之前内容对比讲解]
```

> 💡 技巧：不要每次都开新对话，善用多轮对话让AI理解你的完整需求。

### 技能3：上传文件与图片分析

1. 点击对话框左侧的**📎附件图标**
2. 上传PDF、Word、Excel、图片等文件
3. 提问：「请总结这份报告的核心要点」或「这张图表说明了什么趋势？」

**支持格式：** PDF / Word / Excel / PPT / PNG / JPG / CSV / 代码文件

### 技能4：使用语音对话

1. 点击输入框右侧的**🎙️麦克风图标**
2. 等待AI准备好后开始说话
3. AI会实时语音回复，支持打断和追问

> 适合场景：开车途中学习、语言练习、解放双手的多任务场景

### 技能5：新建对话与管理历史记录

- 点击左侧边栏「**+ New Chat**」开启新对话（避免上下文混淆）
- 历史对话自动保存，可在左侧列表随时回顾
- 支持对对话**重命名**、**分享**、**删除**操作

## 七、新手常犯的5个错误（避坑指南）

1. **❌ 问题太模糊**：「帮我写个文案」→ 应明确产品、受众、场景
2. **❌ 不给背景信息**：AI不了解你的具体情况，回答会很通用
3. **❌ 每次都开新对话**：导致AI无法利用上下文，回答重复
4. **❌ 接受第一次输出**：好的结果往往需要2-3轮迭代打磨
5. **❌ 盲目信任所有输出**：涉及事实、数据、法律的内容务必核实

## 八、FAQ：ChatGPT怎么用常见问题解答

**Q1：ChatGPT怎么用中文？**

ChatGPT天然支持中文，你直接用中文提问即可。如果AI用英文回复，在提示词末尾加上「请用中文回答」或直接告诉它「请用简体中文回复所有问题」。

**Q2：ChatGPT国内没有网络怎么用？**

推荐使用国内镜像平台。[GPTCat](https://gptcat.cc) 和 [SnakeGPT](https://snakegpt.vip) 均可国内直连，无需科学上网工具，用国内邮箱注册即可立即使用。

**Q3：ChatGPT免费版和付费版有什么区别？**

| 功能 | 免费版 | 付费版（Plus/Pro）|
|------|--------|-----------------|
| 模型版本 | GPT-4o（限速） | GPT-5.4（无限制）|
| 使用频率 | 有次数限制 | 大幅提升 |
| 图像生成 | 有限次数 | 更多配额 |
| 高级语音 | 部分支持 | 完整支持 |
| 联网搜索 | 支持 | 支持 |
| 文件上传 | 有限 | 更大容量 |

**Q4：ChatGPT会记住我之前说过的话吗？**

同一对话窗口内，ChatGPT有完整上下文记忆。但如果开启新对话，记忆会重置。付费版支持「Memory（记忆）」功能，可跨对话记住你的偏好信息（需手动开启）。

**Q5：ChatGPT怎么用来写论文？**

ChatGPT可辅助论文写作，但不建议直接"生成论文"。正确用法：
- 用AI**梳理文献综述框架**
- 用AI**解释难懂的学术概念**
- 用AI**润色和优化语言表达**
- 用AI**检查逻辑结构**
- 注意：最终内容需要自己核实数据来源，避免学术不端

**Q6：ChatGPT可以帮我做哪些工作？**

几乎覆盖所有知识型工作：文案写作、代码开发、数据分析、翻译、客服话术、营销策划、学习辅导、法律文件初审、财务分析等。越具体的任务描述，产出越精准实用。

**Q7：ChatGPT输出的内容准确吗？可以直接用吗？**

ChatGPT在语言生成上极为流畅，但**事实类信息可能存在"幻觉"**（即编造不存在的数据或引用）。建议：
- 创意内容、写作框架 → 可直接参考使用
- 事实、数据、法律、医疗 → 务必交叉核实
- 代码逻辑 → 需测试验证
